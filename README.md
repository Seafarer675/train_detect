# 專案簡介

這個專案是一個可以辨識台鐵火車頭型號的系統，使用 YOLO 分類模型 來進行車頭分類，搭配 爬蟲技術 可以取得火車頭的基本資料，並提供 API 供使用者查詢或比較兩個車頭型號。整個系統可以透過 Docker 快速部署。

# 專案特色：

* 支援火車頭型號辨識

* 可透過 API 查詢火車頭資料

* 提供兩個車頭型號比較功能

* Docker 一鍵部署，免繁瑣環境設定

# 功能解說

1. 上傳火車頭圖片 → 返回車型分類結果與信心值

2. 查詢車頭資料（例如製造年份、規格）

3. 比較兩個火車頭的基本資訊

⚠️ 注意：模型權重未上傳至 GitHub。

# 安裝與使用
*本專案已上傳至Docker hub，使用時需先準備好Docker環境*
### 下載鏡像
```
docker pull seafarer45/train_detector
```
### 運行容器
```
docker run -p 8000:8000 train_detector:latest
```
*API 可在 http://localhost:8000/docs#/ 查看*
